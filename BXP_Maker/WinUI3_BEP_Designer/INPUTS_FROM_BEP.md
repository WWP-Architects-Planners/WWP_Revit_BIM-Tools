# Typical BEP Inputs (Extracted from WWP BIM Execution Plan)

Source: `BXP_Maker/WWP_BEP_extracted.txt`

## Project Management Inputs

- Project Number (SvN Internal)
- Project Name
- Project Address
- Project Owner/Client
- Project Type
- Contract Type
- Project Description

## Stakeholder + Responsibility Inputs

- Organization and lead contact
- Discipline lead, project manager, BIM manager, job/design captain
- Contact details (email, timezone, phone)

## Collaboration + ACC Inputs

- ACC Hub project configuration
- Auto-publishing schedule (weekly cadence)
- Sharing timeline (minimum bi-weekly)
- Package naming convention (`<Project>_Shared for <Purpose>`)
- Low-trust workflow toggle (share + consume package control)
- Consumed-folder usage
- Milestone archive paths (`04-PUBLISHED`, `05-ARCHIVED`)

## Clash Detection Inputs

- Clash lead firm
- Clash schedule/milestones
- Required LOD before entering clash workflow
- Required coordination views for ACC
- Clash issue tracking setup
- Session policy:
  - keep current sessions
  - delete from beginning
  - regenerate default sessions

## Standards Inputs

- Revit version/build per discipline
- Linking method (shared coordinates)
- Workset taxonomy and naming
- Model naming conventions
- Shared parameters usage
- Origin/north/georeference references
